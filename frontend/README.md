# SecTestOpsHub Frontend

Frontend uygulamasını local olarak çalıştırmak için:

## Gereksinimler

- Node.js (npm ile birlikte gelir)

## Çalıştırma

```bash
# Frontend'i başlat (otomatik olarak tarayıcıda açar)
npm start

# Veya sadece sunucuyu başlat (tarayıcıyı manuel açarsınız)
npm run serve

# Development modu (cache olmadan, hot reload için)
npm run dev
```

## Erişim

- **Frontend**: http://localhost:8080
- **Backend API**: http://localhost:8000 (Backend'in çalışıyor olması gerekir)

## Notlar

- Backend'in çalışıyor olması gerekiyor (http://localhost:8000)
- Backend'i başlatmak için: `docker compose up` (proje root klasöründen)
